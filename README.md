# OGE Physics Preparation

Материалы для быстрой подготовки к ОГЭ по физике.

Основной срочный документ: [oge_physics_2_day_plan.md](oge_physics_2_day_plan.md).

Подробная версия на 3 дня оставлена как справочник: [oge_physics_3_day_plan.md](oge_physics_3_day_plan.md).

В папке `materials/fipi/` лежат скачанные материалы FIPI:

- страницы FIPI в Markdown;
- демоверсия, спецификация и кодификатор ОГЭ по физике 2026;
- навигатор самостоятельной подготовки по разделам;
- тренировочные задания FIPI.

Шпаргалки в планах предназначены только для домашнего повторения. На экзамен их брать нельзя.
