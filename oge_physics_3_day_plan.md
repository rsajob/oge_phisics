# ОГЭ по физике за 3 дня: сжатый план подготовки

Актуально на 3 июня 2026 года. Если экзамен действительно "через 3 дня", считаем так: день 1 - 3 июня, день 2 - 4 июня, день 3 - 5 июня, экзамен - 6 июня. Если дата другая, просто сдвинуть дни без изменения порядка.

Цель для среднего уровня: сначала стабильно набрать 20+ первичных баллов, затем добирать до 30+ через эксперимент, качественные задачи и одну-две расчетные задачи. По рекомендациям 2026 года по физике: 0-9 - "2", 10-19 - "3", 20-29 - "4", 30-39 - "5".

Важно: "шпаргалки" ниже - это домашние листы для повторения и тренировки. На экзамен их брать нельзя.

## Что надо понимать про экзамен

- В работе 22 задания, максимум 39 первичных баллов, время - 180 минут.
- Можно пользоваться непрограммируемым калькулятором и линейкой.
- Задания 1-16 - краткий ответ, задания 17-22 - развернутый ответ.
- Задание 17 - эксперимент на оборудовании, 3 балла.
- Задания 18-19 - качественные задачи с объяснением, по 2 балла.
- Задания 20-22 - расчетные задачи, по 3 балла.
- В заданиях 6-11 в бланк пишется только число, единицы измерения не писать.
- В заданиях 14 и 16 надо выбрать ровно два утверждения. Лишняя цифра дает 0.

## Главная стратегия на 3 дня

1. Не читать физику подряд. Сначала диагностировать пробелы вариантом.
2. Вести "журнал ошибок": номер задания, тема, причина ошибки, как исправить.
3. Повторять формулы только через задачи: формула -> 3-5 задач -> ошибка -> исправление.
4. Каждый день закрывать 3 слоя: теория, типовые короткие задания, развернутый ответ.
5. Не гнаться за всеми сложными задачами 21-22, пока не держатся 1-17 и 18-20.

## День 1 - диагностика, механика, теплота

### Утро или первый длинный блок: диагностика

1. Пройти демоверсию FIPI 2026 в режиме экзамена или хотя бы за 120-150 минут.
2. Проверить ответы и развернутые задания по критериям.
3. Заполнить таблицу пробелов:

| Линия | Балл | Тема | Ошибка | Действие |
|---:|---:|---|---|---|
| 6 | 0/1 | Механика | Не перевел единицы | 10 задач на перевод и скорость/путь |

Цвета:
- Зеленый - решает быстро и верно.
- Желтый - понимает, но ошибается в единицах, знаках, оформлении.
- Красный - не знает тему или не понимает тип задания.

### Механика - главный блок дня

Линии: 1-7, 12, 14, 19-22.  
Приоритет: движение, силы, давление, Архимед, энергия, импульс, графики.

Нужно закрыть:
- равномерное и равноускоренное движение, графики `v(t)`, `s(t)`, `x(t)`;
- силы: тяжести, трения, упругости, Архимеда;
- давление твердого тела и жидкости;
- работа, мощность, энергия, КПД;
- импульс и закон сохранения импульса;
- простые механизмы: рычаг, блок, наклонная плоскость;
- колебания, волны, звук - коротко, через определения и формулы.

Практика:
- 20-25 коротких задач по линиям 1-7 и 12.
- 5 задач по графикам/таблицам для линии 14.
- 2 качественные задачи линии 19.
- 2 расчетные задачи линии 20 или 21.

### Тепловые явления - быстрые баллы

Линии: 1-5, 8, 12, 14, 19-22.  
Приоритет: формулы количества теплоты, фазовые переходы, тепловой баланс, таблицы.

Нужно закрыть:
- строение вещества, диффузия, броуновское движение;
- теплопередача: теплопроводность, конвекция, излучение;
- нагревание/охлаждение `Q = cmΔt`;
- плавление/кристаллизация `Q = lambda m`;
- испарение/конденсация `Q = Lm`;
- сгорание топлива `Q = qm`;
- уравнение теплового баланса;
- графики нагревания и плавления.

Практика:
- 10 задач на `Q = cmΔt`, `Q = lambda m`, `Q = Lm`, `Q = qm`.
- 5 задач на тепловой баланс.
- 5 задач на таблицы и графики.

## Какие шпаргалки сделать

### Шпаргалка 1. Формулы по разделам

Один лист A4. Формулы группировать по разделам, рядом писать единицы.

### Шпаргалка 2. Единицы и переводы

Обязательно:
- `км/ч -> м/с`: делить на 3,6;
- `м/с -> км/ч`: умножать на 3,6;
- `г -> кг`: делить на 1000;
- `см -> м`: делить на 100;
- `мм -> м`: делить на 1000;
- `мин -> с`: умножать на 60;
- `кДж -> Дж`: умножать на 1000;
- `мА -> А`: делить на 1000;
- `см^3 -> м^3`: умножать на `10^-6`.

### Шпаргалка 3. Электричество и оптика

Сделать схемы:
- амперметр последовательно, вольтметр параллельно;
- последовательное соединение: ток одинаковый, напряжения складываются, сопротивления складываются;
- параллельное соединение: напряжение одинаковое, токи складываются, общее сопротивление меньше меньшего;
- три луча в собирающей линзе.

- 3 пары измерений для исследования зависимости.

### Шпаргалка 5. Качественная задача

Шаблон ответа для 18-19:

Нельзя писать только "потому что так". Нужна опора на закон, формулу или свойство явления.

## Основные формулы

### Механика

- `v = s / t`
- `x = x0 + vx*t`
- `sx = v0x*t + ax*t^2/2`
- `vx = v0x + ax*t`
- `v2^2 - v1^2 = 2*a*s`
- `v = 2*pi*R / T`
- `aц = v^2 / R`
- `nu = 1 / T`
- `rho = m / V`
- `F = m*a`
- `Fтяж = m*g`, где `g = 10 м/с^2`
- `Fтр = mu*N`
- `Fупр = k*delta_l`
- `p = F / S`
- `p = rho*g*h` или `p = rho*g*h + pатм`, если учитывается атмосферное давление
- `Fарх = rho*g*V`
- `A = F*s*cos(alpha)`
- `N = A / t`
- `Ek = m*v^2/2`
- `Ep = m*g*h`
- `E = Ek + Ep`
- `p_имп = m*v`
- `m1*v1 + m2*v2 = const` для замкнутой системы
- `M = F*l`
- `eta = Aполезная / Aзатраченная`
- `lambda_волны = v*T`

### Тепловые явления

- `Q = c*m*(t2 - t1)`
- `Q = lambda*m` для плавления или кристаллизации
- `Q = L*m` для парообразования или конденсации
- `Q = q*m` для сгорания топлива
- `Q1 + Q2 + ... = 0` для теплового баланса
- При нагревании температура растет, при плавлении/кипении температура обычно постоянна, пока идет фазовый переход.

### Электричество, магнетизм, оптика

- `I = q / t`
- `U = A / q`
- `R = rho*l / S`
- `I = U / R`
- Последовательное соединение: `I1 = I2`, `U = U1 + U2`, `R = R1 + R2`
- Параллельное соединение: `U1 = U2`, `I = I1 + I2`, `1/R = 1/R1 + 1/R2`
- Для двух одинаковых параллельных сопротивлений: `Rобщ = R/2`
- `A = U*I*t`
- `P = U*I`
- `Q = I^2*R*t`
- `D = 1 / F` для оптической силы линзы, если `F` в метрах
- Закон отражения: угол падения равен углу отражения.
- Частота света при переходе из одной среды в другую не меняется, скорость и длина волны меняются.

### Квантовые явления

- `A = Z + N`
- `N = A - Z`
- Альфа-распад: массовое число уменьшается на 4, зарядовое число уменьшается на 2.
- Бета-распад: массовое число не меняется, зарядовое число меняется на 1.
- Гамма-излучение: состав ядра не меняется.
- Период полураспада: через каждый период остается половина первоначального количества.

## Алгоритмы для типовых линий

### Задания 1-5

1. Вспомнить раздел: механика, теплота, электричество, оптика, квант.
2. Подчеркнуть физические величины, явления, приборы.
3. Не выбирать по знакомым словам. Проверять смысл определения.

### Задания 6-11

1. Выписать дано.
2. Перевести все в СИ.
3. Выбрать формулу.
4. Подставить.
5. Проверить единицу в условии.
6. В ответ записать только число.

### Задания 12-13

1. Записать формулу связи величин.
2. Что увеличивается? Что уменьшается? Что постоянно?
3. Подставить простые числа, если трудно сравнить в уме.

### Задания 14 и 16

1. Выбрать ровно два утверждения.
2. Каждое утверждение проверять по таблице/графику/опыту, а не по памяти.
3. Не добавлять третью цифру "на всякий случай".

### Задание 17

Главное - прямые измерения. Ошибка в прямом измерении может стоить всех баллов. Делать медленно, чисто, с единицами и понятным рисунком.

### Задания 18-19

Формат:
- ответ;
- закон/явление;
- формула или словесная зависимость;
- применение к условию;
- вывод.

### Задания 20-22

1. Краткое условие с единицами.
2. Справочные величины, если нужны.
3. Основные формулы.
4. Преобразования.
5. Расчет.
6. Ответ с единицей.

Если не получается полностью, писать хотя бы правильные формулы и ход решения - это может дать часть баллов.
